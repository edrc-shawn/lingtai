# 灵台（Lingtai）

> 面向中文知识工作者的 AI 知识引擎。clone → 放原料 → 提炼 → 零到灵。

## 这是什么

灵台是一个基于 Obsidian 的 AI 知识管理系统。核心理念：
- **原料只存不改** — 你的原始内容是真理来源
- **丹房由 AI 增量维护** — AI 提炼、体检、交叉链接
- **零配置启动** — clone 即可用，不绑 AI 平台

## 快速开始

```bash
git clone https://github.com/yourname/lingtai.git 灵台
cd 灵台
# 用 Obsidian 打开此文件夹作为 vault
```

> 然后对 AI 说"提炼"——它会读代码，告诉你该做什么。

## 依赖

- Obsidian（可选，推荐）
- Python 3.10+（跑体检脚本）
- 任意 AI 助手（WB/MiMo/Claude/ChatGPT 均可）

## 零 → 灵

```
第一天：放原料 → 提炼 → 第一颗丹
第一周：每日提炼 → 体检 → 缺口发现
第一月：交叉链接生效 → 图谱渐密 → 台律迭代
```
