# 灵台（Lingtai）

> 把零散笔记变成可检索、会生长的知识库。
> 你只管往里扔笔记，AI 帮你整理、串联、找出缺口。

---

## 看一眼

你下载后会看到一堆文件夹。不用管它们——你只需要知道两个地方：

```
灵台/
├── 原料/    ← 这里，扔你的笔记进去
├── 丹房/    ← AI 整理完的知识会出现在这里
└── 入门/    ← 可选，填一份画像让 AI 更懂你
```

**工作原理**：你往 `原料/` 里扔 Markdown 笔记（一条思考、一篇文章、一段摘录都可以），AI 帮你读、归类、融入 `丹房/` 里的知识页面。扔得越多，知识库越密，越能帮你联想和发现。

## 第一次用（3 步）

### 1. 下载

**[点此下载 ZIP](https://gitee.com/erdong-risheng/lingtai)**（绿色按钮 → 下载 ZIP）→ 解压到任意位置。

如果你会用 git：
```bash
git clone https://github.com/erdong-risheng/lingtai.git
```

### 2. 初始化

打开终端（Windows 按 `Win+R`，输入 `cmd`），进入灵台目录，运行：

```bash
python scripts/lingtai.py init
```

如果提示"python 未找到"，先装 [Python](https://www.python.org/downloads/)（安装时勾选"Add to PATH"）。

初始化会创建好所有目录，30 秒就完。

### 3. 扔笔记 → 喊 AI

**扔笔记**：把你的 Markdown 文件拖进 `原料/` 文件夹。

**喊 AI**：打开你用的 AI 工具（推荐 [WorkBuddy](https://workbuddy.cn)、Claude、ChatGPT），把灵台文件夹作为工作目录，然后对 AI 说：

> 提炼

AI 会自己读规则，知道该怎么做。你等着看 `丹房/` 里长出知识就行。

## 你需要什么

| 东西 | 必须？ | 说明 |
|------|:---:|------|
| Python 3.10+ | ✅ | 跑初始化脚本和体检 |
| AI 助手 | ✅ | 灵台本身不内置 AI，需要你有一个 |
| Obsidian | 🔶 推荐 | 不是必须，但用它看知识图谱会很爽 |
| Git | ❌ | 可以直接下载 ZIP |

**AI 助手推荐**（都有免费额度）：
- [WorkBuddy](https://workbuddy.cn) — 灵台原生支持，自动读台律
- Claude — 把灵台文件夹拖进去即可
- ChatGPT — 上传文件夹或逐个对话

## 接下来会发生什么

```
第1天   → 扔 3 篇笔记，AI 提炼成 1-2 个知识页面
第1周   → 每天扔几篇，知识图谱开始有交叉链接
第1月   → 原料够多，AI 能发现你都没意识到的关联
```

> 灵台不是"更好的文件夹"——它是在你的笔记之间**搭桥**。桥多了，你走到哪都能找到下一块跳板。

## 想让 AI 更懂你？

编辑 `入门/灵台用户画像.md`（初始化时自动生成），填上：
- 你是谁、在做什么
- 你喜欢简洁还是详细
- 你工作的时间段

填了之后，AI 的输出会更贴合你。

---

## 自己读规则？

灵台对 AI 有一套完整操作规范（台律），AI 会自动读。如果你想了解它怎么工作：
- `AGENTS.md` — AI 启动入口
- `台律/运行手册.md` — 完整提炼规则
- `索引.md` — 知识库目录
