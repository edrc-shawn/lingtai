# 灵台新手指引

> 5 分钟，从下载到第一次提炼。不预设任何技术知识。

---

## 第 1 步：下载灵台

打开这个链接 → 点右边绿色「克隆/下载」→「下载 ZIP」：

**https://gitee.com/erdong-risheng/lingtai**

下载完解压，你会得到一个叫 `lingtai` 的文件夹。把它放在你喜欢的位置（桌面也行）。

---

## 第 2 步：装 Python（如果没装过）

按 `Win+R`，输入 `cmd`，回车。然后在黑色窗口输入：

```
python --version
```

- 如果输出 `Python 3.x.x` → ✅ 跳过这一步
- 如果提示"不是内部命令" → 去 https://www.python.org/downloads/ 下载安装。**安装时勾选底部的「Add Python to PATH」**，然后点 Install

---

## 第 3 步：初始化灵台

还是在 cmd 里，进入你的灵台文件夹。比如你解压在桌面：

```
cd Desktop\lingtai
```

然后运行：

```
python scripts\lingtai.py init
```

你会看到一堆 `+ 00-思考与认知`、`+ 原料/` 的输出。看到「灵台已就绪」就完成了。

---

## 第 4 步：扔第一份笔记

打开灵台文件夹，找到 `原料` 文件夹。

把你的 Markdown 笔记拖进去。什么格式的 Markdown 都行——写了一半的文章、从网页摘录的一段话、一个灵感记录。**扔进去就行，不用分类，不用起规范名字。**

> 如果你手头没有 Markdown 文件，先建一个试试：新建文本文件，改名为 `我的第一份笔记.md`，随便写点什么，拖进 `原料/`。

---

## 第 5 步：打开 WorkBuddy，说"提炼"

打开 WorkBuddy，把灵台文件夹设置为工作目录。然后直接说：

> 提炼

AI 会自己读灵台的规则，找到你刚扔的笔记，分析内容，把知识点写入 `丹房/`。

---

## 第 6 步：等着看

提炼完去 `丹房/` 文件夹看看——AI 把笔记变成了知识页面，还自动加了标签和链接。

以后每次有新的笔记，重复第 4 步和第 5 步：扔原料 → 说"提炼"。

---

## 想让 AI 更懂你？（可选）

打开 `入门/灵台用户画像.md`，填几行：

```markdown
## 我是谁
- 身份：自媒体创作者
- 知识领域：商业分析、个人成长
```

填了之后，AI 提炼时会偏向你关注的方向。

---

## 常见问题

**Q: 提炼是干嘛的？**
A: 就是把零散笔记整理成结构化的知识页。一篇笔记里提到 5 个概念，AI 帮你拆开、归到对应的知识领域。

**Q: 提炼完原笔记还在吗？**
A: 在。`原料/` 里的东西只读不删，AI 只在 `丹房/` 里写新的。

**Q: 多久提炼一次？**
A: 随你。每天扔两篇就提取一次，攒一周提炼一次也行。

**Q: 不装 Obsidian 能看结果吗？**
A: 能。WorkBuddy 可以直接预览 markdown 文件。Obsidian 只是让你的知识图谱更好看——装着不碍事，不装也不耽误。
