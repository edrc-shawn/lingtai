## 启动

新 LLM 会话接入后，顺序读：

1. `索引.md` — 全库导航
2. `台律/运行手册.md` — 通用规则
3. `台律/结构.md` — 格式规范
4. `台律/系统架构.md` — 全系统文件关联图谱（改东西前先读）
5. 按触发词选择入口（见下方）

## 触发词 → 入口

| 触发 | 作用 | 入口文件 |
|:----|:----|:--------|
| 提炼 / ingest | 原料→丹房 | `台律/运行手册.md` + `台律/结构.md` |
| 问知 / query | 跨域检索+合成 | `索引.md` + 目标丹房页 |
| 体检 / lint | 全库健康检查 | `python .workbuddy/scripts/lint_check.py` |
| 每日检 | 08:00 自动巡检 | `体检/自动化/巡更·每日检.md` |
| 每周查 | 周一 09:00 自动查 | `体检/自动化/巡更·每周查.md` |
| 每月问 | 1日 10:00 自动查 | `体检/自动化/巡更·每月问.md` |
| 内观 | 22:00/手动生成 | `台律/运行手册.md §4.7` |
| 产出 | 公众号/小红书/抖音 | 见对应 skill 文件 |
