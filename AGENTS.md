# AGENTS.md — 灵台知识库入口

> 本文件是灵台对 LLM 的入口索引。不含规则——规则在台律/。

## 这是什么

灵台（Lingtai），基于 Obsidian 的 AI 知识管理系统。原料只存不改，丹房由 AI 增量维护。

## 启动流程

```
会话开始：
  □ 1. Read 索引.md ——了解仓库结构
  □ 2. Read 台律/说明.md ——了解规则体系
  □ 3. Read 台律/运行手册.md ——获取操作规范
  □ 4. Read 台律/结构.md ——获取格式规范
  □ 5. 开工

会话结束（有产出时）：
  □ 按运行手册 4.2 走完工清单（入库体检→时间线→索引→日志→Git）
```

## 目录结构

```
├── 台律/          ← 根本法：运行手册 + 结构 + 标签 + 变更评估 + 说明
├── 丹房/          ← Wiki 层：AI 提炼的知识库
│   ├── 0-入门/    ← 入门指南
│   ├── 00-07/     ← 领域分类
│   ├── 99-一人公司/ ← 个人事业
│   └── 词典/      ← 核心概念
├── 体检/          ← 巡检：仪表盘 + 对账 + 待审 + 内观 + 体检规格
├── 原料/          ← AI 只读不改
├── 作品/          ← 已发布内容
├── scripts/       ← 体检/日志/语义探针工具脚本
├── 索引.md        ← 全库导航（含 ⚡🔒 标记规则）
└── 日志.md        ← 操作日志
```

## 规则去哪找
